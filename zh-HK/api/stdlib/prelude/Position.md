# Position

表格和標籤在圖表面板上的位置。

**種類:** enum

**成員**

| 名稱 | 說明 |
| --- | --- |
| `TopLeft` | 左上角。 |
| `TopCenter` | 頂部居中。 |
| `TopRight` | 右上角。 |
| `MiddleLeft` | 中部左側。 |
| `MiddleCenter` | 正中央。 |
| `MiddleRight` | 中部右側。 |
| `BottomLeft` | 左下角。 |
| `BottomCenter` | 底部居中。 |
| `BottomRight` | 右下角。 |
