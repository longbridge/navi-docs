# SettlementAsClose

Whether a futures symbol's close is the settlement price on `1D`+.

**Kind:** enum

**Variants**

| Name | Description |
| --- | --- |
| `Inherit` | Use the chart symbol's default setting. |
| `On` | Use the settlement price as close. |
| `Off` | Use the actual closing price. |
