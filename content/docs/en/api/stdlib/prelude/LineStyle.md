# LineStyle

Style of a line drawing.

**Kind:** enum

**Variants**

| Name | Description |
| --- | --- |
| `Solid` | Solid line. |
| `Dashed` | Dashed line. |
| `Dotted` | Dotted line. |
| `ArrowLeft` | Solid line with left-facing arrow. |
| `ArrowRight` | Solid line with right-facing arrow. |
| `ArrowBoth` | Solid line with arrows on both ends. |
