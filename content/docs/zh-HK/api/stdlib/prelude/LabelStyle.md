# LabelStyle

標籤繪圖的樣式。

**種類:** enum

**成員**

| 名稱 | 說明 |
| --- | --- |
| `NoLabel` | 無標籤形狀。 |
| `ArrowDown` | 向下箭頭標籤。 |
| `ArrowUp` | 向上箭頭標籤。 |
| `Circle` | 圓形標籤。 |
| `Cross` | 十字標籤。 |
| `Diamond` | 菱形標籤。 |
| `Flag` | 旗形標籤。 |
| `LabelCenter` | 居中標籤框。 |
| `Down` | 向下指向的標籤框。 |
| `Left` | 向左指向的標籤框。 |
| `LowerLeft` | 向左下指向的標籤框。 |
| `LowerRight` | 向右下指向的標籤框。 |
| `Right` | 向右指向的標籤框。 |
| `Up` | 向上指向的標籤框。 |
| `UpperLeft` | 向左上指向的標籤框。 |
| `UpperRight` | 向右上指向的標籤框。 |
| `Square` | 方形標籤。 |
| `TextOutline` | 文字輪廓標籤。 |
| `TriangleDown` | 向下三角形標籤。 |
| `TriangleUp` | 向上三角形標籤。 |
| `XCross` | X 形標籤。 |
