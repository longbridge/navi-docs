# Position

表格和标签在图表面板上的位置。

**种类:** enum

**成员**

| 名称 | 说明 |
| --- | --- |
| `TopLeft` | 左上角。 |
| `TopCenter` | 顶部居中。 |
| `TopRight` | 右上角。 |
| `MiddleLeft` | 中部左侧。 |
| `MiddleCenter` | 正中央。 |
| `MiddleRight` | 中部右侧。 |
| `BottomLeft` | 左下角。 |
| `BottomCenter` | 底部居中。 |
| `BottomRight` | 右下角。 |
