# LabelStyle

标签绘图的样式。

**种类:** enum

**成员**

| 名称 | 说明 |
| --- | --- |
| `NoLabel` | 无标签形状。 |
| `ArrowDown` | 向下箭头标签。 |
| `ArrowUp` | 向上箭头标签。 |
| `Circle` | 圆形标签。 |
| `Cross` | 十字标签。 |
| `Diamond` | 菱形标签。 |
| `Flag` | 旗形标签。 |
| `LabelCenter` | 居中标签框。 |
| `Down` | 向下指向的标签框。 |
| `Left` | 向左指向的标签框。 |
| `LowerLeft` | 向左下指向的标签框。 |
| `LowerRight` | 向右下指向的标签框。 |
| `Right` | 向右指向的标签框。 |
| `Up` | 向上指向的标签框。 |
| `UpperLeft` | 向左上指向的标签框。 |
| `UpperRight` | 向右上指向的标签框。 |
| `Square` | 方形标签。 |
| `TextOutline` | 文字轮廓标签。 |
| `TriangleDown` | 向下三角形标签。 |
| `TriangleUp` | 向上三角形标签。 |
| `XCross` | X 形标签。 |
