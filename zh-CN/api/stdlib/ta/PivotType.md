# ta.PivotType

`ta.pivot_point_levels` 所使用的枢轴点计算方法。

**种类:** enum

**成员**

| 名称 | 说明 |
| --- | --- |
| `Traditional` | 标准 HLC/3 枢轴，具有对称的支撑/阻力位。 |
| `Fibonacci` | 斐波那契比例枢轴水平。 |
| `Woodie` | 以开盘价加权的枢轴。 |
| `Classic` | 与传统枢轴相同。 |
| `DM` | 德马克枢轴（非对称，取决于开盘价与收盘价的关系）。 |
| `Camarilla` | 卡马利亚窄幅区间水平。 |
